turbsim_defaultBins - Contains the bins with wind speed and TI as defined for Humbolt Bay

turbsim_increasedTI - Conatins the bins with same wind speed as above, TI has been increased to account for wake effects using Frandsen model - distance between FWTS set as 9.6D

turbsim_increasedTI_redVel - Conatins the bins with reduced wind speed using Jennsen model, TI has been increased to account for wake effects using Frandsen model - distance between FWTS set as 9.6D
